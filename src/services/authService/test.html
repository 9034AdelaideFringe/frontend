Important Notes:

  • The API sets an HttpOnly, SameSite=Strict cookie named jwt for session management.
  • HttpOnly cookies are not accessible via JavaScript (document.cookie), so they won't appear in the "Cookie 状态" section below. This is a security feature.
  • SameSite=Strict means this HTML page must be served from the same domain as the API (https://x.badtom.dpdns.org) for the jwt cookie to be sent with subsequent requests. If opened as a local file (file://) or from a different domain, authentication will likely fail after login.
  • The "清除所有Cookie" button cannot clear HttpOnly cookies. Use the "登出" button or browser developer tools.
